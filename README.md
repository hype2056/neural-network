# Neural Network from Scratch

A feedforward neural network implemented in pure NumPy to recognize handwritten digits from the MNIST dataset.

## What's built from scratch
- Forward pass
- ReLU and softmax activation functions
- Backpropagation
- Gradient descent

## Results
89% accuracy on 41,000 training images after 500 iterations

## Tech
Python, NumPy, Matplotlib

## Run
1. Download `train.csv` from Kaggle's Digit Recognizer dataset
2. Place it in the project folder
3. Run `python neural_network.py`
